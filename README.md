# Danex Digital

A Tailwind-powered React website rebuilt from the original `DanexDigital.html` landing page.

## Available Scripts

- `npm install` - install dependencies
- `npm run dev` - start development server
- `npm run build` - build production files
- `npm run preview` - preview the production build
- `npm run typecheck` - run TypeScript type checking

## Project Structure

- `src/App.tsx` - application root and route configuration
- `src/pages/Homepage.tsx` - main homepage content
- `src/pages/Services.tsx` - services page
- `src/components/Header.tsx` - responsive navigation header
- `src/components/Footer.tsx` - footer with quick links and CTA
